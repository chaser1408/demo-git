* chú thích: - wd: working directory - sa: staging area - lp: local repository 1. git init: tạo ra local repo(.git) 2. git status: kiểm tra xem những thay đổi của code (để biết được file hiện đang nằm ở vùng nào) 3. git add tên_file: chuyển 1 file từ vùng wd sang vùng sa. và git add . là áp dụng cho tất cả file 4. git rm --cached tên_file: đưa 1 file từ vùng sa quay ngược lại vùng wd 5. git commit -m "tin nhắn": đưa tất cả file có sự thay đổi từ vùng sa sang lp, bắt buộc phải có tin nhắn. 6. config git account: git config --global user.name "Your Name" 7. config git email account: git config --global user.email "you@example.com" 8. git log: kiểm tra lịch sử commit 9. git remote add origin remote-repo-url: tạo kết nữa lp với remote repo 10. git push origin master: đẩy code dưới lp lên remote repo (áp dụng trong trường ten_branch)
